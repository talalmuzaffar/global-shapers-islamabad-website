## Image Guidelines

- **Quality**: Use high-resolution images (at least 1000px on the longest side)
- **Optimization**: Compress images before uploading to reduce file size
- **Format**: 
  - Use JPG for photos
  - Use PNG for logos/graphics with transparency
  - Use SVG for illustrations/icons
- **Aspect Ratios**:
  - Logo: Landscape (3:1 or 4:1)
  - Hero images: Landscape (4:3 or 16:9)
  - Project images: Portrait (3:4)
  - Event images: Landscape (4:3)
